Build a strong foundation in Data Structures and Algorithms using Python.

Target:

- Complete MySirG DSA Course (50 Lectures + 28 Assignments)

# Phase 1: Learning DSA

Duration: Until Course Completion

## Lecture Workflow

For EVERY lecture:

1. Watch lecture.
2. Write code yourself.
3. Run code.
4. Modify code.
5. Break code intentionally.
6. Fix errors.
7. Complete related assignment.
8. Write notes.

NEVER:

Lecture → Lecture → Lecture → Lecture

ALWAYS:

Lecture → Coding → Assignment → Notes → Next Lecture

# When to Start LeetCode

Start AFTER learning:

- Arrays
- Strings
- Hashing
- Linked Lists
- Stacks
- Queues

Then begin:

- Easy Problems Only

Avoid Medium and Hard initially.

# Problem Solving Framework

Whenever solving a problem:

Step 1:
Understand Input and Output.

Step 2:
Create a small example.

Step 3:
Write algorithm in plain English.

Step 4:
Convert algorithm into Python.

Step 5:
Test with multiple inputs.

Step 6:
Optimize later.

Correctness first.
Optimization later.
